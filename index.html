NotesApp

En desarrollo avanzado, solo falta despliegue del proyecto.

Se trata de una aplicación web full-stack desarrollada con React.js en el front-end,Node.js en el back-end y que además utiliza PostgreSQL como base de datos. Esta aplicación web permite crear usuarios, iniciar sesión, crear, editar y eliminar notas con un máximo de 255 caracteres. Para su implementación se utilizó JWT token ,Express ,Sequelize ,Dotenv y Git.
Cabe destacar que la aplicación cuenta con un diseño responsivo, lo que permite que pueda ser accedida desde cualquier dispositivo, ya sea computadora, laptop, celular o tablet.

Weather App

El proyecto es un Weather App, realizado utilizando React.js y Tailwind CSS para el frontend y además consume los datos de una API externa. Esta app web permite a los usuarios obtener información meteorológica en tiempo real de diferentes ciudades del mundo. La interfaz cuenta con una barra inferior que permite al usuario navegar las distintas secciones, además se puede buscar ciudades del mundo así como también detectar la ubicación del usuario y mostrar el pronóstico de su ubicación. Una vez que se ha ingresado la ciudad, la aplicación web mostrará información meteorológica detallada, incluyendo la temperatura actual, la humedad, la velocidad del viento, la dirección del viento y la previsión del tiempo para los próximos días. En resumen, esta app web será una herramienta útil y práctica para aquellos que deseen estar informados sobre el clima en diferentes partes del mundo.

Clima app

El proyecto es un Weather App, realizado utilizando React.js y CSS para el frontend y ademas consume los datos de una API externa. Esta app web permite a los usuarios obtener información meteorológica en tiempo real de diferentes ciudades del mundo. La interfaz de usuario cuenta con una barra de búsqueda en la parte superior, donde el usuario podrá ingresar el nombre de la ciudad de la que desea obtener información meteorológica. Una vez que se ha ingresado la ciudad, la aplicación web mostrará información meteorológica detallada, incluyendo la temperatura actual, la humedad, la velocidad del viento, la dirección del viento y la previsión del tiempo para los próximos días. En resumen, esta app web será una herramienta útil y práctica para aquellos que deseen estar informados sobre el clima en diferentes partes del mundo.

Ta te ti

El proyecto web consiste en un juego de tateti (también conocido como tres en raya) utilizando HTML, CSS y JavaScript para el frontend. El juego permitirá a dos jugadores competir entre sí en el mismo dispositivo, con un jugador eligiendo las fichas X y el otro las fichas O La página web muestra la información necesaria durante el juego, como el turno actual, el resultado del juego y un botón de reinicio para comenzar una nueva partida. También se incluirá una sección de puntuación para mostrar el número de partidas ganadas por cada jugador.
